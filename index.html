About the website

This website presents data on alcohol consumption trends among Australians over several years.
It features interactive visualizations, allowing users to toggle between bar and line charts in order to explore how drinking habits have changed over time.
The site is designed for those interested in public health, providing a clear and accessible way to analyze national alcohol consumption patterns.
